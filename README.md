## 🌟 Hi there! I'm Rose Lee 👋

저는 **프론트엔드 개발자**로, **React**와 **TypeScript**를 활용한 사용자 친화적인 인터페이스 제작에 열정을 가지고 있습니다.  
효율적인 코드와 매끄러운 사용자 경험을 만들기 위해 노력하며, 성장하는 개발자가 되는 것이 목표입니다.

- **데이터베이스:** Supabase, MongoDB
